KEY FINDING:

  which should suppress the Header and Footer. However, both the <banner> and
  <contentinfo> elements are rendered and visible at all viewports and themes.

  The form itself, logo, inputs, buttons, mode-switch link, and responsive
  layout (no horizontal scroll) are all functioning correctly.

  This is likely a layout configuration issue — the auth variant is not
  properly excluding the Header/Footer components from the render tree.
